# PROCESSUS – recrutement et d’onboarding collaborateurs

## 1. Objet et finalité  

Ce processus a pour finalité d’assurer l’attraction, la sélection, l’embauche et l’intégration efficace des nouveaux collaborateurs chez Neo Financia. Il garantit une expérience fluide, conforme aux obligations légales, culturelles et de sécurité interne, tout en favorisant l’engagement et la performance dès l’arrivée.

## 2. Champ d’application / Ressources Humaines & Culture  

Ce processus s’applique à l’ensemble des recrutements internes (CDI, CDD, alternants, stages) réalisés pour les entités de Neo Financia en France et à l’international, ainsi qu’au parcours d’intégration (onboarding) jusqu’à la fin de la période d’essai.

## 3. Parties prenantes & RACI (rôles)  

| Activité                        | RH | Manager recruteur | DSI | Juridique | Sécurité |
|-------------------------------|----|-------------------|-----|-----------|----------|
| Expression du besoin          | A  | R                 |    |           |          |
| Publication de l’offre        | R  | A                 |    |           |          |
| Sélection des candidats       | R  | A                 |    |           |          |
| Validation de l'embauche      | R  | A                 |    | C         | C        |
| Création du contrat           | A  |                   |    | R         |          |
| Préparation onboarding IT     | C  |                   | A  |           | R        |
| Intégration jour J + suivi    | R  | A                 | C  |           | C        |

R = Réalisé par / A = Autorité de validation / C = Consulté

## 4. Entrées, déclencheurs et fournisseurs (SIPOC – S)  

- **Entrées** : fiche de poste, budget validé, besoin exprimé par le manager  
- **Déclencheurs** : création de poste, remplacement, plan de croissance  
- **Fournisseurs** : manager opérationnel, contrôle de gestion, prestataires de recrutement, systèmes RH

## 5. Macro-workflow BPMN / chaîne de valeur  

### 5.1 Vue d’ensemble (diagramme texte + numérotation)

1. Identification du besoin & validation du poste  
2. Rédaction et diffusion de l’offre  
3. Réception, tri et sélection des candidatures  
4. Conduite des entretiens et évaluation  
5. Choix du candidat et validation RH / Direction  
6. Création du contrat et formalités administratives  
7. Préparation logistique (accès IT, badge, équipement)  
8. Accueil & parcours d’intégration (onboarding)  
9. Suivi de la période d’essai

### 5.2 Sous-processus clés  

- **Sourcing multicanal** : jobboards, cooptation, cabinets  
- **Entretien structuré** : grille d’évaluation, scoring  
- **Onboarding digitalisé** : plateforme e-learning, buddy, checklist IT & RH  
- **Suivi post-intégration** : points à 1 mois et 3 mois, feedback à chaud

### 5.3 Points de contrôle & jalons  

- Validation budgétaire avant publication  
- Double validation managériale avant offre  
- Signature électronique du contrat  
- Vérification conformité RGPD & clause de confidentialité  
- Suivi onboarding + clôture période d’essai

## 6. Sorties et clients internes/externes (SIPOC – O/C)  

- **Sorties** : collaborateur onboardé, dossier RH complet, accès IT provisionné  
- **Clients internes** : manager, équipe projet, fonctions support  
- **Clients externes** : cabinet de recrutement, organismes d’alternance

## 7. Ressources & systèmes support (IT, data, fournisseurs)  

- ATS (Applicant Tracking System) – ex : TalentSoft, SmartRecruiters  
- SIRH / Core RH – ex : Workday  
- Signature électronique – ex : DocuSign  
- Portail d’onboarding digital (RH + IT)  
- Fournisseurs : cabinets de chasse, prestataires de tests techniques

## 8. Exigences de conformité (REGS)  

- Code du travail & RGPD (UE 2016/679)  
- ISO 9001 (gestion RH)  
- ISO 27001 (gestion des accès IT)  
- Politique sécurité Neo Financia (charte IT, clause NDA)  
- Règlement intérieur & convention collective applicable

## 9. Indicateurs de performance (KPIs) & seuils SLA  

- Délai moyen de recrutement (objectif : ≤ 45 jours)  
- Taux de satisfaction onboarding (objectif : ≥ 85 %)  
- Taux de transformation entre offre et acceptation (objectif : ≥ 90 %)  
- Taux de réussite période d’essai (objectif : ≥ 95 %)  
- Nombre d’erreurs administratives (objectif : 0)

## 10. Risques, gaspillages (Lean) et actions de mitigation  

| Risque / Gaspillage              | Action correctrice                                 |
|----------------------------------|-----------------------------------------------------|
| Retard de contractualisation     | Automatisation + signature électronique             |
| Mauvaise affectation de matériel| Checklist IT centralisée partagée                   |
| Intégration mal perçue          | Buddy + parcours e-learning personnalisé            |
| Erreur de paie                  | Double vérification fiche RH + paie                 |
| Données personnelles mal protégées | Portail sécurisé, rôle du DPO intégré              |

## 11. Interfaces & dépendances croisées (processus amont/aval)  

- Amont : planification RH, validation budgétaire, mobilité interne  
- Aval : gestion des accès, formation continue, performance RH  
- Dépendances : DSI (accès et équipements), juridique (contrat), sécurité (badge)

## 12. Plans d’amélioration continue (PDCA)  

- Plan : analyse des retours candidats et managers après intégration  
- Do : ajustement du parcours d’onboarding tous les semestres  
- Check : revue semestrielle des KPIs recrutement/onboarding  
- Act : mise à jour des supports de formation et outils digitaux
